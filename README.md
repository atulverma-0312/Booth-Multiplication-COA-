# Booth Multiplication Algorithm (COA Project)

##  Description
This project implements Booth’s Multiplication Algorithm using Scilab.

## Features
- Handles signed numbers
- Uses 2’s complement
- Efficient multiplication

## Algorithm
- Check (Q0, Q-1)
- Perform Add/Subtract
- Arithmetic Right Shift

## How to Run
1. Open Scilab
2. Run .sce file
3. Enter inputs

## Sample Output
Multiplicand = 4  
Multiplier = -5  
Result = -20
